ThreadMeister — Heat-Set Insert Holes

ThreadMeister automates the creation of heat-set insert holes in Fusion 360 using CNC Kitchen's tested insert dimensions.

Quick Start

  1. Create a sketch and place sketch points where insert holes should go.
  2. Finish the sketch.
  3. Open SOLID → MODIFY → ThreadMeister.
  4. Select the target body.
  5. Select one or more sketch points.
  6. Choose insert size and hole type (blind or through).
  7. Enable/disable chamfer and bottom fillet (both recommended).
  8. Click OK.

Chamfer & Depth

The top chamfer (default 0.5 mm × 45°) cuts into the bore, reducing usable depth for the insert. The blind_hole_extra_depth setting (default 1.0 mm) compensates for this and provides clearance below the insert.

Default: 1.0 mm extra depth − 0.5 mm chamfer = 0.5 mm clearance below the insert.

Configuration

Edit config.ini in the add-in folder to customize chamfer size, extra depth, bottom fillet radius, and insert definitions. Custom inserts can be added in the [Inserts] section.
